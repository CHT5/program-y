Download

The code for Program-Y is hosst on github, you are probably already on GitHub which is why you are reading this document, but the specific link is https://github.com/keiffster/program-y

If you don't know GitHub or just want to see the code without contributing, click on the 'Clone or Download' button and this will download a .zip file of all the code as it currently stands, taking directly from the HEAD of master on the code base.

Installation

Once you have the code downloaded, unzip into the directory of you choice and then switch to this directory. You should see a directory structure similar to \bots \docs \src And various files

Installtion for OS-X

  1. Install python 3 from: https://www.python.org/downloads/
  2. Install libraries :
    1. pip3 install pyyaml
    2. pip3 install python-dateutil
    3. pip3 install requests

Running a bot

All bots are located in individual folders in \bots folder. Different bots exist for different reasons. I would suggest starting with alice2, the core bot created by the originators of AIML.

Change to this folder and execute the script alice2.sh.

This will start a console version of the bot and you will be presented with a welcome message and a prompt '>>>'.

You can now start interacting with Alice2 AIML grammar. Ask her questions.... go on its fun
